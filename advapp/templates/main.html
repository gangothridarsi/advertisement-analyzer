About Advertisement Analyzer

Allows the user to create new advertisements and also displays statistics collectively for all their ads.

Features

Create New Ads

Users can create new ads from the create section.

Display Ads

Users can view available ads from display section.

Reactions Collection

Users reactions are collected through face recognition and are stored in database.

Display Statistics

Users can see all the reactions for the ads through a dashboard displaying the percent for each reaction.

Frequently Asked Questions

  • How to use Advertisement Analyzer?

    Advertisement Analyzer is very simple to use. After logging to your account, you can either create a new advertisement or view all the available advertisements. By clicking the button when you view an advertisement, your expression will be detected and stored in a database.

  • What features does Advertisement Analyzer have?

    Currently Advertisement Analyzer supports Display/Manage/Detect/Results features. We are starving more to add many new exciting features for our users and enhance experience.

  • Is Advertisement Analyzer free and safe?

    Yes Advertisement Analyzer is free as well as safe for all the users. Your personal information is never shared with anyone, also the emotion at the time of detection is secured and safe.
